## Getting started

1. Clone this repo and enter!

   ```bash
   git clone https://github.com/llere-alt/elit.git
   cd elit/
   ```

2. Install dependencies.

   ```bash
   cd client && npm i
   cd ../server && npm i
   ```

3. Start the app.

- on the server folder
  ```bash
  nodemon index.js
  ```
- on the client folder
  ```bash
  npm start
  ```
